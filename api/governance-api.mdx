---
title: GovernanceApi
description: Access staking, validator, and governance information
---

The GovernanceApi provides methods for querying staking information, validator details, and network governance state. All methods are in the `suix` namespace.

## Methods

### getStakesByIds

Returns one or more `DelegatedStake` objects by their IDs. If a stake was withdrawn, its status will be `Unstaked`.

<ParamField path="staked_sui_ids" type="ObjectID[]" required>
  Array of StakedSui object IDs
</ParamField>

<ResponseField name="result" type="DelegatedStake[]">
  Array of delegated stake information
  
  <Expandable title="DelegatedStake fields">
    <ResponseField name="stakedSuiId" type="ObjectID">The StakedSui object ID</ResponseField>
    <ResponseField name="stakingPool" type="ObjectID">The staking pool ID</ResponseField>
    <ResponseField name="validatorAddress" type="SuiAddress">Validator address</ResponseField>
    <ResponseField name="principal" type="string">Principal staked amount</ResponseField>
    <ResponseField name="stakeRequestEpoch" type="string">Epoch when stake was requested</ResponseField>
    <ResponseField name="stakeActiveEpoch" type="string">Epoch when stake became active</ResponseField>
    <ResponseField name="estimatedReward" type="string">Estimated reward amount</ResponseField>
    <ResponseField name="status" type="string">Stake status: Active, Pending, or Unstaked</ResponseField>
  </Expandable>
</ResponseField>

```bash
curl -X POST https://fullnode.mainnet.sui.io:443 \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "suix_getStakesByIds",
    "params": [
      ["0x..."]
    ]
  }'
```

<ParamField path="epoch" type="BigInt<u64>">

  <Expandable title="response fields">

</ResponseField>

## Usage Example with Rust SDK

```rust
use sui_sdk::SuiClientBuilder;
use sui_types::base_types::SuiAddress;
use std::str::FromStr;

#[tokio::main]
async fn main() -> Result<(), anyhow::Error> {
    let sui = SuiClientBuilder::default()
        .build("https://fullnode.mainnet.sui.io:443")
        .await?;
    
    // Get system state
    let system_state = sui.governance_api()
        .get_latest_sui_system_state()
        .await?;
    
    println!("Current Epoch: {}", system_state.epoch);
    println!("Reference Gas Price: {}", system_state.reference_gas_price);
    println!("Total Stake: {}", system_state.total_stake);
    println!("Active Validators: {}", system_state.active_validators.len());
    
    // Get reference gas price
    let gas_price = sui.governance_api()
        .get_reference_gas_price()
        .await?;
    
    println!("Gas Price: {} MIST", gas_price);
    
    // Get stakes for an address
    let address = SuiAddress::from_str("0x...")?;
    let stakes = sui.governance_api()
        .get_stakes(address)
        .await?;
    
    for stake in stakes {
        println!("Stake: {} SUI with validator {}", 
            stake.principal, 
            stake.validator_address);
    }
    
    // Get validator APY
    let apys = sui.governance_api()
        .get_validators_apy()
        .await?;
    
    println!("Validator APYs for epoch {}", apys.epoch);
    
    Ok(())
}
```
