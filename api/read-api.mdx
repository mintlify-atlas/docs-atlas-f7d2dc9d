---
title: ReadApi
description: Query objects, transactions, checkpoints, and network state
---

The ReadApi provides read-only access to Sui network data. All methods are in the `sui` namespace.

## Methods

### getTransactionBlock

Returns transaction details by digest.

<ParamField path="digest" type="TransactionDigest" required>
  The digest of the queried transaction
</ParamField>

<ParamField path="options" type="SuiTransactionBlockResponseOptions">
  Options for specifying the content to be returned
  
  <Expandable title="options fields">
    <ParamField path="showInput" type="boolean">Show transaction input data</ParamField>
    <ParamField path="showEffects" type="boolean">Show transaction effects</ParamField>
    <ParamField path="showEvents" type="boolean">Show emitted events</ParamField>
    <ParamField path="showObjectChanges" type="boolean">Show object changes</ParamField>
    <ParamField path="showBalanceChanges" type="boolean">Show balance changes</ParamField>
  </Expandable>
</ParamField>

<ResponseField name="result" type="SuiTransactionBlockResponse">
  The transaction response object
</ResponseField>

```bash
curl -X POST https://fullnode.mainnet.sui.io:443 \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "sui_getTransactionBlock",
    "params": [
      "7nPJKfZTx2YWqZWW3A8cXnFQVFQs7FKqBZqXU6qz5K4Q",
      {
        "showInput": true,
        "showEffects": true,
        "showEvents": true
      }
    ]
  }'
```

### getObject

Returns object information for a specified object.

<ParamField path="object_id" type="ObjectID" required>
  The ID of the queried object
</ParamField>

<ParamField path="options" type="SuiObjectDataOptions">
  Options for specifying the content to be returned
  
  <Expandable title="options fields">
    <ParamField path="showType" type="boolean">Show object type</ParamField>
    <ParamField path="showOwner" type="boolean">Show object owner</ParamField>
    <ParamField path="showPreviousTransaction" type="boolean">Show previous transaction digest</ParamField>
    <ParamField path="showContent" type="boolean">Show parsed object content</ParamField>
    <ParamField path="showBcs" type="boolean">Show BCS-encoded bytes</ParamField>
    <ParamField path="showDisplay" type="boolean">Show display metadata</ParamField>
    <ParamField path="showStorageRebate" type="boolean">Show storage rebate</ParamField>
  </Expandable>
</ParamField>

<ResponseField name="result" type="SuiObjectResponse">
  The object information
</ResponseField>

```bash
curl -X POST https://fullnode.mainnet.sui.io:443 \
  -H "Content-Type: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "sui_getObject",
    "params": [
      "0x5",
      {
        "showType": true,
        "showOwner": true,
        "showContent": true
      }
    ]
  }'
```

### verifyZkLoginSignature

Verifies a zkLogin signature for the given bytes, intent scope, and author.

<ParamField path="bytes" type="string" required>
  Base64 string of BCS bytes for raw transaction data or personal message
</ParamField>

<ParamField path="signature" type="string" required>
  Base64 string of the zkLogin signature to verify
</ParamField>

<ParamField path="intent_scope" type="ZkLoginIntentScope" required>
  Intent scope: either `TransactionData` or `PersonalMessage`
</ParamField>

<ParamField path="author" type="SuiAddress" required>
  The author of the signature
</ParamField>

<ResponseField name="result" type="ZkLoginVerifyResult">
  Verification result
</ResponseField>
