## What You'll Build

In this quickstart, you'll:
- Install Sui CLI and tools
- Create your first Move package
- Write a simple smart contract
- Deploy to Sui devnet
- Interact with your contract

<Note>
This tutorial takes approximately **10 minutes** to complete.
</Note>

<AccordionGroup>

## Step 1: Install Sui

<Steps>
  <Step title="Install Rust">
    If you don't have Rust installed:
    
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    source "$HOME/.cargo/env"
    ```
  </Step>

  <Step title="Install Sui CLI">
    Install Sui from the main branch:
    
    ```bash
    cargo install --locked --git https://github.com/MystenLabs/sui.git --branch main sui
    ```
    
    This takes 5-10 minutes to compile.
    
    <Tip>
    For faster installation, download pre-built binaries from the [releases page](https://github.com/MystenLabs/sui/releases).
    </Tip>
  </Step>

  <Step title="Verify Installation">
    ```bash
    sui --version
    ```
    
    Expected output: `sui 1.68.0` or later
  </Step>

  <Step title="Configure Sui Client">
    Initialize your Sui configuration:
    
    ```bash
    sui client
    ```
    
    This creates a new keypair and connects to devnet by default.
  </Step>

  <Step title="Get Test Tokens">
    Request SUI tokens from the devnet faucet:
    
    ```bash
    sui client faucet
    ```
    
    Wait ~10 seconds, then verify:
    
    ```bash
    sui client balance
    ```
  </Step>
</Steps>

## Step 2: Create Your First Move Package

<Steps>
  <Step title="Initialize New Package">
    ```bash
    sui move new hello_world
    cd hello_world
    ```
    
    This creates a new Move package with this structure:
    ```
    hello_world/
    ├── Move.toml
    └── sources/
        └── hello_world.move
    ```
  </Step>

  <Step title="Write Your Smart Contract">
    Open `sources/hello_world.move` and replace with:
    
    ```move
    module hello_world::hello {
        use std::string::{Self, String};
        
        /// A simple object that holds a message
        public struct HelloWorld has key {
            id: UID,
            message: String,
        }
        
        /// Create a new HelloWorld object
        public fun create(message: vector<u8>, ctx: &mut TxContext) {
            let hello = HelloWorld {
                id: object::new(ctx),
                message: string::utf8(message),
            };
            transfer::share_object(hello);
        }
        
        /// Update the message
        public fun update(hello: &mut HelloWorld, message: vector<u8>) {
            hello.message = string::utf8(message);
        }
        
        /// Get the current message
        public fun get_message(hello: &HelloWorld): String {
            hello.message
        }
    }
    ```
  </Step>

  <Step title="Build the Package">
    ```bash
    sui move build
    ```
    
    If successful, you'll see:
    ```
    BUILDING hello_world
    Build Successful
    ```
  </Step>
</Steps>

## Step 3: Deploy to Devnet

<Steps>
  <Step title="Publish the Package">
    ```bash
    sui client publish --gas-budget 100000000
    ```
    
    <Info>
    The gas budget (0.1 SUI) covers the cost of publishing the package.
    </Info>
  </Step>

  <Step title="Save Package ID">
    After publishing, you'll see output like:
    
    ```
    ╭──────────────────────────────────────────────────╮
    │ Object Changes                                    │
    ├──────────────────────────────────────────────────┤
    │ Published Objects:                                │
    │  ┌──                                             │
    │  │ PackageID: 0xabcd1234...                      │
    │  └──                                             │
    ╰──────────────────────────────────────────────────╯
    ```
    
    **Save this PackageID** - you'll need it to interact with your contract!
  </Step>
</Steps>

## Step 4: Interact with Your Contract

<Steps>
  <Step title="Call the Create Function">
    Replace `PACKAGE_ID` with your actual package ID:
    
    ```bash
    sui client call \
      --package PACKAGE_ID \
      --module hello \
      --function create \
      --args "Hello, Sui!" \
      --gas-budget 10000000
    ```
  </Step>

  <Step title="Find the Object ID">
    The transaction will create a new `HelloWorld` object. Look for:
    
    ```
    Created Objects:
      ┌──
      │ ObjectID: 0x5678abcd...
      │ Type: PACKAGE_ID::hello::HelloWorld
      └──
    ```
    
    Save this **ObjectID**.
  </Step>

  <Step title="Update the Message">
    Update the message in your object:
    
    ```bash
    sui client call \
      --package PACKAGE_ID \
      --module hello \
      --function update \
      --args OBJECT_ID "Hello from Sui blockchain!" \
      --gas-budget 10000000
    ```
  </Step>

  <Step title="Verify the Update">
    Query your object to see the updated message:
    
    ```bash
    sui client object OBJECT_ID
    ```
    
    You'll see the object data including your message!
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Build fails with 'package not found'">
    Ensure you're in the package directory:
    ```bash
    cd hello_world
    sui move build
    ```
  </Accordion>

  <Accordion title="Faucet request fails">
    Try the web faucet: https://faucet.devnet.sui.io/
    
    Or switch environments:
    ```bash
    sui client switch --env testnet
    sui client faucet
    ```
  </Accordion>

  <Accordion title="Publish fails with 'InsufficientGas'">
    Increase the gas budget:
    ```bash
    sui client publish --gas-budget 200000000
    ```
    
    Or get more test tokens from the faucet.
  </Accordion>

  <Accordion title="Command not found: sui">
    Add Cargo bin to PATH:
    ```bash
    export PATH="$HOME/.cargo/bin:$PATH"
    echo 'export PATH="$HOME/.cargo/bin:$PATH"' >> ~/.bashrc
    ```
  </Accordion>
</AccordionGroup>
